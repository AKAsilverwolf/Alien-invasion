# 📁 项目文件作用说明

## 🎮 游戏核心文件

### `alien_invasion.py` - 主程序入口
- **作用**: 游戏的启动入口，包含主游戏循环
- **功能**: 初始化Pygame、创建游戏对象、处理主循环、事件分发、背景音乐控制

### `settings.py` - 游戏设置管理
- **作用**: 存储和管理所有游戏配置参数
- **功能**: 屏幕设置、速度参数、难度配置、动态难度调整、外星人生成设置

### `ship.py` - 飞船类
- **作用**: 实现玩家控制的飞船
- **功能**: 四向移动、生命值管理(3条生命)、无敌状态(1秒无敌)、视觉闪烁效果

### `alien.py` - 外星人类
- **作用**: 实现敌对外星人
- **功能**: 随机生成(从顶部随机位置)、随机速度(0.3-2.0)、边缘反弹AI、碰撞检测

### `bullet.py` - 子弹类
- **作用**: 实现子弹系统
- **功能**: 子弹移动、碰撞检测、自动清理超出屏幕的子弹、最大3发限制

### `game_functions.py` - 游戏函数库
- **作用**: 包含各种游戏逻辑函数
- **功能**: 事件处理(包含M键音乐控制)、碰撞检测、外星人生成、游戏状态更新、键盘控制

### `game_stats.py` - 游戏统计
- **作用**: 管理游戏状态和统计数据
- **功能**: 游戏状态管理、生命值追踪(3条生命)、分数统计、等级管理、最高分加载

### `scoreboard.py` - 计分板显示
- **作用**: 实时显示游戏信息
- **功能**: 分数显示、生命显示、等级显示、排行榜显示、中文字体支持

### `button.py` - 按钮组件
- **作用**: 通用按钮UI组件
- **功能**: 按钮渲染、点击检测、中文字体支持、Start Game按钮实现

### `leaderboard.py` - 排行榜管理
- **作用**: 排行榜数据管理
- **功能**: 数据存储(JSON格式)、排序、持久化、UTF-8中文支持、高分判断

### `name_input.py` - 名字输入界面
- **作用**: 高分后的名字输入界面
- **功能**: 中文输入、键盘处理、UI绘制、确认取消功能

### `sound_manager.py` - 音效管理器
- **作用**: 管理所有音效和背景音乐
- **功能**: 音效文件加载(shoot.wav, explosion.wav, BGM.mp3)、播放控制、音量调节、M键音乐控制、程序化音效备份

### `resource_manager.py` - 资源路径管理器
- **作用**: 统一管理资源文件路径
- **功能**: 路径解析、开发/打包环境适配(sys._MEIPASS处理)、异常处理、图像/音效/数据路径获取

## 🛠️ 工具和打包脚本

### `build_game.py` - 主打包脚本
- **作用**: Python实现的完整打包脚本，跨平台支持
- **功能**: 清理旧文件、执行打包、错误处理、进度显示、文件验证、资源检查

### `build_sound_fixed.bat` - Windows完整打包脚本
- **作用**: Windows批处理打包脚本（音效修复版）
- **功能**: 一键打包、中文提示、音效修复配置、资源包含、文件大小显示

### `create_portable.bat` - 便携版创建器
- **作用**: 创建完整的可分发便携版
- **功能**: 复制所有文件、生成启动脚本(开始游戏.bat)、创建说明文件(游戏说明.txt)、完整验证

### `AlienInvasion.spec` - PyInstaller配置文件
- **作用**: PyInstaller的详细配置
- **功能**: 打包参数、资源包含、路径配置、自动生成配置文件

## 📁 资源文件夹

### `images/` - 图像资源文件夹
- **作用**: 存储所有游戏图像
- **内容**:

- **作用**: 存储所有音效文件
- **内容**:

- **作用**: 存储打包生成的可执行文件
- **内容**: `AlienInvasion.exe`等打包结果(约10-20MB)

- **作用**: 便携版游戏完整包
- **内容**: exe文件、启动脚本、说明文件、资源文件
