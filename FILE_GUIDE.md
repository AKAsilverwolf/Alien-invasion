# 📁 项目文件作用说明

## 🎮 游戏核心文件

### `alien_invasion.py` - 主程序入口
- **作用**: 游戏的启动入口，包含主游戏循环
- **功能**: 初始化Pygame、创建游戏对象、处理主循环、事件分发
- **重要性**: ⭐⭐⭐⭐⭐ (核心文件，删除游戏无法运行)

### `settings.py` - 游戏设置管理
- **作用**: 存储和管理所有游戏配置参数
- **功能**: 屏幕设置、速度参数、难度配置、动态难度调整
- **重要性**: ⭐⭐⭐⭐⭐ (核心文件，删除游戏无法正常运行)

### `ship.py` - 飞船类
- **作用**: 实现玩家控制的飞船
- **功能**: 四向移动、生命值管理、无敌状态、视觉闪烁效果
- **重要性**: ⭐⭐⭐⭐⭐ (核心文件，删除游戏无法运行)

### `alien.py` - 外星人类
- **作用**: 实现敌对外星人
- **功能**: 随机生成、随机速度、边缘反弹AI、碰撞检测
- **重要性**: ⭐⭐⭐⭐⭐ (核心文件，删除游戏无敌人)

### `bullet.py` - 子弹类
- **作用**: 实现子弹系统
- **功能**: 子弹移动、碰撞检测、自动清理超出屏幕的子弹
- **重要性**: ⭐⭐⭐⭐⭐ (核心文件，删除无法射击)

### `game_functions.py` - 游戏函数库
- **作用**: 包含各种游戏逻辑函数
- **功能**: 事件处理、碰撞检测、外星人生成、游戏状态更新
- **重要性**: ⭐⭐⭐⭐⭐ (核心文件，删除游戏逻辑混乱)

### `game_stats.py` - 游戏统计
- **作用**: 管理游戏状态和统计数据
- **功能**: 游戏状态、生命值、分数、等级管理
- **重要性**: ⭐⭐⭐⭐ (重要文件，删除无法追踪游戏进度)

### `scoreboard.py` - 计分板显示
- **作用**: 实时显示游戏信息
- **功能**: 分数显示、生命显示、等级显示、渲染管理
- **重要性**: ⭐⭐⭐ (重要文件，删除不影响游戏但影响UI)

### `button.py` - 按钮组件
- **作用**: 通用按钮UI组件
- **功能**: 按钮渲染、点击检测、文本显示
- **重要性**: ⭐⭐⭐ (重要文件，删除影响界面交互)

### `leaderboard.py` - 排行榜管理
- **作用**: 排行榜数据管理
- **功能**: 数据存储、排序、持久化、JSON操作
- **重要性**: ⭐⭐⭐ (重要文件，删除影响排行榜功能)

### `name_input.py` - 名字输入界面
- **作用**: 高分后的名字输入界面
- **功能**: 中文输入、键盘处理、UI绘制
- **重要性**: ⭐⭐ (中等重要，删除不影响基本游戏)

### `sound_manager.py` - 音效管理器
- **作用**: 管理所有音效和背景音乐
- **功能**: 音效加载、播放控制、音量调节、M键控制
- **重要性**: ⭐⭐⭐⭐ (重要文件，删除无声音)

### `resource_manager.py` - 资源路径管理器
- **作用**: 统一管理资源文件路径
- **功能**: 路径解析、开发/打包环境适配、异常处理
- **重要性**: ⭐⭐⭐⭐ (重要文件，删除打包后无法加载资源)

## 🛠️ 工具和打包脚本

### `build_game.py` - 主打包脚本
- **作用**: Python实现的完整打包脚本
- **功能**: 清理旧文件、执行打包、错误处理、进度显示
- **重要性**: ⭐⭐⭐ (开发工具，删除仍可手动打包)

### `build_sound_fixed.bat` - Windows完整打包脚本
- **作用**: Windows批处理打包脚本（音效修复版）
- **功能**: 一键打包、中文提示、音效修复配置
- **重要性**: ⭐⭐⭐ (开发工具，用户友好)

### `quick_build.bat` - 快速打包脚本
- **作用**: 简化的快速打包工具
- **功能**: 快速生成测试版本、简化输出
- **重要性**: ⭐⭐ (开发工具，开发时方便)

### `create_portable.bat` - 便携版创建器
- **作用**: 创建完整的可分发便携版
- **功能**: 复制所有文件、生成启动脚本、创建说明
- **重要性**: ⭐⭐⭐ (分发工具，用户友好)

### `create_images.py` - 图像生成脚本
- **作用**: 程序化生成游戏图像
- **功能**: 生成飞船、外星人、背景图像
- **重要性**: ⭐⭐⭐ (重要工具，删除丢失图像需手动创建)

### `AlienInvasion.spec` - PyInstaller配置文件
- **作用**: PyInstaller的详细配置
- **功能**: 打包参数、资源包含、路径配置
- **重要性**: ⭐⭐ (配置文件，可自动生成)
