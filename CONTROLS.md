# 🎮 游戏控制说明

## 🕹️ 基本控制

| 控制方式 | 功能说明 | 详细说明 |
|---------|---------|---------|
| **方向键 (↑↓←→)** | 控制飞船四向移动 | 上键向上移动，下键向下移动，左键向左移动，右键向右移动 |
| **空格键** | 发射子弹 | 最多同时3发子弹，自动超出屏幕清理 |
| **L键** | 查看排行榜 | 游戏中随时可用，ESC键返回 |
| **M键** | 暂停/恢复背景音乐 | 游戏中切换音乐播放状态 |
| **ESC键** | 关闭排行榜/退出输入界面 | 从排行榜或名字输入界面返回游戏 |
| **Q键** | 快速退出游戏 | 立即退出游戏 |
| **鼠标点击** | 点击按钮操作 | 用于点击Start Game按钮 |

## 🎵 音效控制

### M键音乐控制
- **功能**: 切换背景音乐播放状态
- **操作**: 游戏中按下M可以暂停或恢复背景音乐
- **反馈**: 控制台会显示"背景音乐已暂停"或"背景音乐已恢复"
- **智能控制**: 
  - 打开排行榜或输入名字时音乐会自动暂停
  - 返回游戏时音乐会自动恢复

### 音效系统说明
- **🎵 背景音乐**: 循环播放，营造游戏氛围
- **💥 射击音效**: 发射子弹时播放
- **💫 爆炸音效**: 击毁外星人时播放
- **🏁 游戏结束音效**: 游戏结束时播放

## 🎮 游戏机制控制

### 生命值系统
- **3条生命**: 玩家开始时拥有3条生命
- **无敌机制**: 被击中后获得1秒无敌时间
- **视觉反馈**: 无敌状态时飞船会闪烁
- **失去生命**: 被外星人撞击会失去一条生命

### 分数系统
- **基础分数**: 每击毁一个外星人获得基础分
- **等级加成**: 等级越高，分数奖励越多
- **连续击杀**: 快速连续击杀有额外奖励
- **分数显示**: 实时显示在屏幕左上角

### 排行榜系统
- **自动记录**: 游戏结束时自动记录高分
- **前10名**: 只保留前10名高分记录
- **中文支持**: 完美支持中文名字输入和显示
- **持久存储**: 排行榜数据本地保存，重启不丢失

## 🔧 控制设置

### 系统要求
- **操作系统**: Windows/macOS/Linux
- **输入设备**: 键盘 + 鼠标
- **音频设备**: 扬声器或耳机(推荐)

### 自定义选项
可通过修改 `settings.py` 来自定义：
- 飞船移动速度: `ship_speed_factor`
- 子弹速度: `bullet_speed_factor`
- 子弹数量限制: `bullets_allowed`
- 外星人速度范围: `alien_speed_min`, `alien_speed_max`
- 音效音量: `sound_volume`

## 🐛 控制问题排查

### 常见问题
1. **方向键不响应**: 检查键盘是否连接正常
2. **L键无反应**: 确认不在其他输入界面中
3. **M键音乐无效**: 检查音频设备是否正常连接

### 故障排除
1. **键盘输入无响应**: 重新启动游戏，检查键盘连接
2. **鼠标点击无效**: 确认鼠标在游戏窗口内
3. **音效控制异常**: 检查系统音频设备设置
4. **中文显示问题**: 确保系统安装了中文字体

## 📋 快速参考

### 快捷键速查
```
↑↓←→  - 移动飞船
空格  - 发射子弹
L     - 查看排行榜  
M     - 暂停/恢复音乐
ESC   - 返回/退出
Q     - 退出游戏
鼠标  - 点击按钮
```

### 游戏状态控制
```
主菜单  - 鼠标点击Start Game开始
游戏中  - 方向键+空格键控制
游戏中  - L键查看排行榜
游戏中  - M键控制音乐
游戏中  - Q键退出
排行榜  - ESC键返回
名字输入 - ESC键返回游戏
```

### 音效控制流程
```
游戏启动 → 自动播放背景音乐
按M键   → 暂停背景音乐
再按M键 → 恢复背景音乐
打开排行榜 → 自动暂停音乐
返回游戏  → 自动恢复音乐
```
