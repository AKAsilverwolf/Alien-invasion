# 🎮 游戏控制说明

## 🕹️ 基本控制

| 控制方式 | 功能说明 |
|---------|---------|
| **方向键 (↑↓←→)** | 控制飞船四向移动 |
| **空格键** | 发射子弹 |
| **L键** | 查看排行榜（游戏中随时可用） |
| **M键** | 暂停/恢复背景音乐 |
| **ESC键** | 关闭排行榜/退出输入界面 |
| **Q键** | 快速退出游戏 |
| **鼠标点击** | 点击按钮操作 |

## 🎵 音效控制

- **M键**: 切换背景音乐播放状态
  - 游戏中按下M可以暂停或恢复背景音乐
  - 打开排行榜或输入名字时音乐会自动暂停
  - 返回游戏时音乐会自动恢复

## 🔊 音效说明

游戏包含以下音效：
- **🎵 背景音乐**: 循环播放，营造游戏氛围
- **💥 射击音效**: 发射子弹时播放
- **💫 爆炸音效**: 击毁外星人时播放
- **🏁 游戏结束音效**: 游戏结束时播放

## 🎯 游戏技巧

1. **音效提示**: 听音效可以帮助判断游戏状态
2. **音乐控制**: 不喜欢背景音乐时按M键关闭
3. **节奏配合**: 配合背景音乐的节奏进行游戏
